Description of Service

Internet Speed Test provides a free, browser-based internet speed testing tool. The service measures:

  • Download speed — how fast data reaches your device
  • Upload speed — how fast your device sends data
  • Ping (latency) — round-trip response time to a server
  • Jitter — variation in ping over time

All tests run in your browser. No software download or installation is needed.

Accuracy of Results

Speed test results are estimates based on your current connection conditions. Actual speeds may vary depending on:

  • Network congestion and time of day
  • Wi-Fi signal strength versus wired Ethernet connection
  • Your ISP's network routing and throttling policies
  • Distance from the test server
  • Other devices and applications using your network

Results should be used as a general indicator of connection quality, not as a guaranteed measurement. For the most accurate results, test over a wired connection with other applications closed.
